DevZest .Net Licensing a lightweight yet comprehensive licensing solution for Microsoft .Net platform.
Protect your .Net components and applications from illegal use.

Activate this library by executing LicenseConsole.exe under the package "tools" folder,
and then obtain a 30 days free trial evaluation license, or a purchased product license.

See Documentation.chm for more information.

Samples source code available at:
https://github.com/DevZest/Licensing.Samples
