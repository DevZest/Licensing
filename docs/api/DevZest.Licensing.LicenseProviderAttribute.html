Syntax
public abstract class LicenseProviderAttribute : Attribute, _Attribute
Remarks

License provider is declared as assembly level attribute. It loads a License object from a specified location.

When initialized during validation, the license provider is sealed by setting its IsFrozen property to true. Any attempting to modify the LicenseProviderAttribute object throws an System.InvalidOperationException. All derived classes should respect this rule by calling VerifyFrozenAccess() in all public methods and property setters that might change the object.
